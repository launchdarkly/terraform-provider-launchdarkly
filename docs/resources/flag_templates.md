# launchdarkly_flag_templates (Resource)

Manages the Custom flag-template settings for a LaunchDarkly project.

## Example Usage

```terraform
resource "launchdarkly_flag_templates" "example" {
  project_key = "my-project"

  tags      = ["terraform"]
  temporary = false

  boolean_defaults = [{
    true_display_name  = "True"
    false_display_name = "False"
    true_description   = ""
    false_description  = ""
    on_variation       = 0
    off_variation      = 1
  }]
}
```
